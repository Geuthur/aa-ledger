Ledger Guide

Character Ledger shows all Ratting / ESS Payout / Mining entrys per Month / Day
You can switch between Months to show what you have already done and how much it was.


If you want more information about the Income you can use the Information Button on the Right side.


Corporation Ledger shows all TAX / ESS TAX per Month / Day
Note: All taxes of each alts are listed under the name of the main.


FAQ

Q: I only see my Main on Corp Ledger.
A: The Amounts already include all Alts!

Q: I don't see my Character on the Char Ledger.
A: Maybe your Character isn't in the System, Please add it!


Q: Why is the ESS Stolen Amount not correctly
A: The ESS Stolen information are updated from 'Corporation Ledger' so it can be wrong, cause it has a 3 hours update period

Q: I don't see any ESS Payout
A: This can happen if your Corporation hasn't add The Corporation to the Ledger
